🔐 Password Generator

A simple, customizable password generator built with vanilla HTML, CSS, and JavaScript. Choose your password length (4–64 characters) and pick which character types to include — uppercase, lowercase, numbers, and special characters — then generate a random password and copy it to your clipboard with one click.

Features:

Adjustable password length
Toggle uppercase, lowercase, numbers, and special characters
Input validation (warns if no character type is selected or length is out of range)
One-click copy to clipboard with confirmation message
Clean, minimal UI, no dependencies or build tools required
